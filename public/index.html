A list of the exposed endpoints on e-missions api


GET https://final-project-phase.herokuapp.com/api/

You are here, viewing this page


GET https://final-project-phase.herokuapp.com/api/users

Returns a list of all users


GET https://final-project-phase.herokuapp.com/api/users/:email

Returns one user based on email


GET https://final-project-phase.herokuapp.com/api/users/:email/journeys

Returns all user's journeys


POST https://final-project-phase.herokuapp.com/api/users

Creats a user


GET https://final-project-phase.herokuapp.com/api/journeys

Returns all journeys


GET https://final-project-phase.herokuapp.com/api/journeys/journey_id

Returns a journey by id


POST https://final-project-phase.herokuapp.com/api/journeys

Creates a journey
